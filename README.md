# Title

Workout Buddy

# Description

App that allows user to create their own workout routines.

# Features

- Allow users to add a workout
- Allow users to delete a workout

# How to use

1. cd backend
2. npm run dev
3. cd frontend
4. npm start

# Technologies

- React
- JavaScript
- HTML
- CSS
- Node
- Express
- MongoDB
